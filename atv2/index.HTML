Projetos

Projeto - TaskFlow

João Pedro desenvolveu diversos projetos que demonstram sua versatilidade como programador. Um dos destaques é o TaskFlow, um gestor de tarefas online criado para ajudar usuários a organizarem suas atividades de maneira eficiente. Utilizando React no frontend, Node.js no backend e MongoDB como banco de dados, o TaskFlow conta com uma interface responsiva e recursos modernos, como o arrastar e soltar (drag-and-drop), além de autenticação segura com JWT.

Projeto - ShopSmart

Outro projeto notável é o ShopSmart, uma loja virtual completa que oferece desde a exibição de produtos até a integração com sistemas de pagamento, como Stripe e PayPal. Construído com Django e PostgreSQL, o ShopSmart inclui um painel administrativo para que vendedores possam gerenciar produtos, estoques e pedidos, sendo totalmente otimizado para dispositivos móveis.

Projeto - API Clima Fácil

João também desenvolveu a API Clima Fácil, uma solução que fornece informações climáticas em tempo real. Essa API, criada com Flask e integrada à OpenWeather API, é facilmente escalável graças ao uso do Docker. Além disso, a API conta com uma documentação detalhada, permitindo que outros desenvolvedores a utilizem com facilidade em seus projetos.
